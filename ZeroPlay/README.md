# Chronicle of the Idle Hero

Willkommen bei "Chronicle of the Idle Hero", einem in Python geschriebenen Idle-RPG mit einer grafischen Benutzeroberfläche, die mit Tkinter erstellt wurde. In diesem Spiel erschaffst du einen Helden, schickst ihn auf Quests und siehst zu, wie er stärker wird, Beute sammelt und aufsteigt, auch wenn du nicht aktiv spielst.

## Features

*   **Idle-Gameplay:** Starte eine Quest und dein Held wird sie selbstständig abschließen. Mit dem **Auto-Quest-Modus** kannst du das Spiel im Hintergrund laufen lassen und dein Held wird kontinuierlich neue Abenteuer beginnen.
*   **Charakterentwicklung:**
    *   Wähle aus drei einzigartigen Klassen: **Krieger**, **Magier** oder **Schurke**.
    *   Steige im Level auf, um deine Basisattribute zu verbessern.
    *   Sammle Erfahrungspunkte (XP) durch das Abschließen von Quests.
*   **Umfassendes Beutesystem:**
    *   Finde Ausrüstung mit verschiedenen Seltenheitsstufen, von "Schlecht" bis "Mythisch".
    *   Rüste Gegenstände in den Slots Kopf, Brust und Waffe aus.
    *   Jeder Gegenstand hat einen Seltenheitswert, der durch seine Farbe angezeigt wird, und Boni auf deine Hauptattribute.
    *   Upgrade-Anzeigen (⭐) zeigen dir sofort, welche Gegenstände besser sind als deine aktuell ausgerüsteten.
*   **Inventar- und Händlersystem:**
    *   Verwalte dein Inventar und rüste neue Gegenstände per Knopfdruck aus.
    *   Benutze Verbrauchsgüter wie Heil- und Manatränke.
    *   Besuche den Händler, um nicht benötigte Gegenstände zu verkaufen und nützliche Tränke zu kaufen.
*   **Speichersystem:** Dein Fortschritt wird automatisch gespeichert, wenn du das Spiel schließt. Du kannst deinen Helden jederzeit über das Hauptmenü laden.
*   **Dynamische Benutzeroberfläche:**
    *   Detaillierte Tooltips im Inventar zeigen dir die Werte eines Gegenstandes, wenn du mit der Maus darüber fährst.
    *   Eine saubere und übersichtliche Oberfläche, die dir alle wichtigen Informationen auf einen Blick liefert.

## Wie man spielt

1.  **Starte das Spiel:** Führe die Datei `main.py` aus.
2.  **Neues Spiel oder Laden:** Im Hauptmenü kannst du ein neues Spiel starten oder einen bestehenden Spielstand laden.
3.  **Charakter erstellen:** Wähle eine der drei Klassen und gib deinem Helden einen Namen.
4.  **Quests beginnen:** Klicke auf "Neue Quest beginnen", um ein Abenteuer zu starten, oder aktiviere den "Auto-Quest"-Modus, damit dein Held selbstständig Quests erledigt.
5.  **Ausrüstung verwalten:** Behalte dein Inventar im Auge. Rüste bessere Gegenstände aus, um die Attribute deines Helden zu verbessern.
6.  **Händler besuchen:** Wenn dein Inventar voll ist oder du Tränke benötigst, besuche den Händler zwischen den Quests.

## Ausführen des Spiels

Um das Spiel zu starten, führe einfach das Hauptskript aus:

```bash
python main.py
```
